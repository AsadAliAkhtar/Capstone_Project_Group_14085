# Notebooks

This folder contains exploratory and reporting notebooks. Reusable functions are implemented in `src/`, while notebooks provide the ordered execution workflow.

Recommended order:

1. `01_data_extraction.ipynb`
2. `02_data_wrangling.ipynb`
3. `03_hmm_experiments.ipynb`
4. `04_msvar_experiments.ipynb`
5. `05_backtesting.ipynb`

The final backtest and reporting outputs are generated by `05_backtesting.ipynb`.
