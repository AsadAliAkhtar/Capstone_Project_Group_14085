# Data

This folder contains the monthly macroeconomic and market datasets used in the project.

Files:

- `macro_data.csv`: aligned macroeconomic series downloaded from FRED before final transformation.
- `macro_clean.csv`: transformed model-ready macroeconomic features.
- `market_data.csv`: monthly asset price and return data before final cleaning.
- `market_clean.csv`: model-ready monthly asset return series for equities, bonds, and gold.

All datasets are aligned to a common month-end date index.
